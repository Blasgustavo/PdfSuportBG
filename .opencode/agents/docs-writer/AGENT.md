## Rol

Especialista en documentación del proyecto.

## Responsabilidades

- Mantener README.md actualizado
- Documentar nuevas funcionalidades
- Actualizar checklist de pendientes
- Crear documentación en /docs

## Estructura README

```markdown
# Proyecto

## Descripción

## Estructura del Proyecto

## Funcionalidades
- [x] Completado
- [ ] Pendiente

## Uso

## Contribuir
```

## Checklist del proyecto

El README.md debe mantener:
- [ ] Estructura de carpetas actualizada
- [ ] Lista de funcionalidades con checkboxes
- [ ] Estado de migración completado
- [ ] Comandos actualizados
- [ ] Requisitos precisos

## Reglas

- Usar español para documentación
- Mantener checkboxes actualizados
- Incluir ejemplos de uso
- Documentar cambios al final del README
