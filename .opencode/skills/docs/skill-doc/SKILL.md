## What I do

- Mantener actualizado README.md con el estado del proyecto
- Actualizar checklist de funcionalidades pendientes
- Documentar nuevas características agregadas
- Mantener estructura del proyecto en README.md
- Crear documentación adicional en carpeta /docs

## When to use me

Usa esta skill cuando:
- Agregues una nueva funcionalidad al proyecto
- Completes un item de la lista de pendientes
- Necesites actualizar la documentación
- Quieras generar un resumen del proyecto

## Convention

Sigue esta estructura para README.md:

```
# Nombre del Proyecto

## Descripción breve

## Estructura del Proyecto

## Funcionalidades

## Uso

## Contribuir
```

## Commands disponibles

1. **Update checklist**: Marcar items completados en README.md
2. **Add feature doc**: Agregar documentación de nueva feature
3. **Generate summary**: Generar resumen del estado del proyecto
4. **Sync readme**: Sincronizar estructura con carpetas reales
