## What I do

- Verificar estado del repositorio con `git status`
- Mostrar cambios pendientes con `git diff`
- Actualizar lista de tareas pendientes del proyecto
- Verificar si hay actualizaciones de dependencias
- Sincronizar estructura del proyecto según README.md

## When to use me

Usa esta skill cuando:
- Necesites ver el estado actual del proyecto
- Quieras sincronizar cambios antes de hacer un commit
- Necesites verificar que la estructura del proyecto está actualizada
- Antes de iniciar una nueva sesión de trabajo

## Commands disponibles

1. **Ver estado**: Ejecutar `git status` y mostrar cambios
2. **Ver diff**: Mostrar diferencias pendientes
3. **Actualizar checklist**: Verificar items completados en README.md
4. **Sync structure**: Comparar estructura real con README.md
